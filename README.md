# Проект: Место

**О проекте**

Учебный проект позволяющий получить практические навыки по технологиям,  
использования специализированного ПО.


В этой версии реализованны:

* Закрытие popup
  * Кликом на фон
  * Кнопкой ESC
* Валидация форм для "Новое место" и "Данных профиля"

**Технологии**

* HTML
* CSS
    * @media
    * @font-face
    * Flexbox
    * Grid
    * Псевдоклассы
* БЭМ (Nested)
    * Наименование классов
    * Файловая система по БЭМ
* JS  
  * Events
  * Keyword event
  * Click event
  * Custom validation form  

**ПО**

* VSCode
* IntelliJ IDEA
* Figma (PC, Web)
* Git
    * branch
    * stash  
    * merge
